## Research Preparation Workflow

### Phase 1: Understand the Research Task
1. file_viewer and analyze the input idea.json thoroughly
2. Identify the core research problem, proposed method, and expected contributions

### Phase 2: Comprehensive Literature & Resource Research

Use `search_web` and `browse_web` to gather the following information:

#### A. Comparable Papers & Baselines
- Search for recent papers on the same topic (last 2-3 years)
- Identify state-of-the-art methods in this domain
- Find the original papers that introduced key concepts

#### B. Datasets & Benchmarks
- Search for standard datasets used in this research area
- Find official download links (HuggingFace, paper repositories, institutional websites)
- Note dataset size, format, preprocessing requirements
- Identify synthetic data generation scripts if applicable
- **CRITICAL**: For each dataset, you MUST find and document its benchmark/leaderboard

#### C. Evaluation Metrics
- List all standard metrics used in this field (accuracy, F1, RMSE, BLEU, etc.)
- Find domain-specific metrics (physics-informed losses, conservation errors, etc.)
- Note how metrics should be computed and reported

#### D. Baseline Performance (CRITICAL - DO NOT SKIP)
- Record reported performance numbers from comparable papers
- **For each method, record performance on EACH dataset**
- Create a structured performance table in idea.md
- Note the experimental setup (hardware, epochs, batch size, etc.)
